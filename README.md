# Luing-Heritage-Webmap
Webmap containing maps, hillshade and vector data for Luing Island
## Created June 2026

# Made for Luing History Group

Webmap with layers showing 0.5 m Digital Terrain Mode depicted as hillshade and Relief Vidualisation Layer generated from LiDAR captured in 2022. OS Six Inch 1888 - 1913 map (NLS Map images - National Library of Scotland - Licence CC BY 4.0). National Record of the Historic Environment sites for Luing downloaded from Trove (https://www.trove.scot)
